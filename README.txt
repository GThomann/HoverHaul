Repositorio criado para aplicar, testar e aperfeiçoar habilidades de desenvolvimento no Unity, em um único projeto.

A idéia inicial do tema do projeto surgiu em uma conversa informal durante uma aula e começou a ser implementada como trabalho da disciplina de "Jogos para console", na FATEC-Americana.

Os objetivos principais do projeto são:
-Divertir-se enquanto o projeto evolui;
-Não ter limite para a criatividade - toda idéia é valida;
-Aprender todos os aspectos do desenvolvimento - criação e arte, programação e documentação (GDD, engenharia de software e legislativo);
-Desenvolvimento colaborativo a distância.
